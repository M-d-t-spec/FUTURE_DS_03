# FUTURE_DS_03

## Project Overview

The dashboard and analysis focus on:
- **Overall Satisfaction Trends**: Shows how students rated events 1-5 to understand general satisfaction.
- **Categorical Analysis**: Compares Organization, Speaker quality, Venue, and willingness to attend again.
- **Text Feedback & Sentiment**: Classifies open ended responses (Liked_Most, Improvements, Additional_Comments) into Positive, Neutral, and Negative sentiment.
- **Actionable Recommendations**: Uses data insights to suggest improvements for future events.

## Dataset

- **CSV File:** college_event_feedback.csv containing:
- Column
- Description
- Event name
- Name of the event
- Overall rating
- Organization
- Rating of event organizations
  and many more
